Группы:
  USER - обычные участники голосования
  ADMIN - администраторы

Права доступа:

| Действие                     | USER | ADMIN |
|------------------------------|------|-------|
| Просмотр голосований         |  ✔   |   ✔   |
| Участие в голосовании        |  ✔   |   ✔   |
| Создание голосований         |  ✘   |   ✔   |
| Редактирование голосований   |  ✘   |   ✔   |
| Удаление голосований         |  ✘   |   ✔   |
| Управление тегами            |  ✘   |   ✔   |
| Просмотр результатов         |  ✔   |   ✔   |
| Управление пользователями    |  ✘   |   ✔   |

Реализация:
  UserService.isMemberOfGroup(User user, String groupName)
    - true, если пользователь состоит в группе с указанным именем
    - используется для проверки прав перед операциями
